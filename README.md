# AWR Stream Hub - Frontend Ultra Pro

واجهة Android Jetpack Compose تجريبية لتطبيق أفلام ومسلسلات وأنمي مع:
- Home سينمائي
- Movies / K-Drama / Anime
- Search
- Requests
- Detail sheet
- زر Fetch Subtitles لكل عمل
- UI placeholders للباك إند القادم من GitHub JSON

Build:
```bash
gradle :app:assembleDebug --stacktrace
```
